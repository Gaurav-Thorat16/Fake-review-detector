# 🕵️ Fake Review Detector

A smart web application that detects **fake product reviews** using Natural Language Processing (NLP) and machine learning techniques.
The system analyzes review text and classifies it as **Real** or **Fake** to help users make better purchasing decisions.

---

## 🚀 Project Overview

Online platforms like Amazon, Flipkart, and Google Reviews often contain **manipulated or fake reviews** intended to influence buyers.
This project aims to **automatically detect suspicious reviews** by analyzing the language patterns and sentiment of the review text.

The system provides a **dashboard interface** where users can input reviews and instantly see whether they are genuine or fake.

---

## 🧠 Key Features

* 🔍 Detects **fake vs real reviews**
* 📊 Simple **dashboard interface**
* ⚡ Fast review analysis
* 💬 Supports **multiple review inputs**
* 🧠 Uses **NLP-based classification**

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Gaurav-Thorat16/Fake-review-detector.git
```

Navigate to the project folder:

```
cd Fake-review-detector
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```

Open in browser:

```
http://localhost:5173
```
